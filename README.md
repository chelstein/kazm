# KAZM 106.5 FM & 780 AM

Mellow Mountain Radio

Serving Sedona, the Verde Valley, and Northern Arizona with Yacht Rock, local news, weather, sports, community information, and live radio programming.

## Features

* Live Audio Streaming
* Programming Schedule
* Local News
* Weather
* Sports Coverage
* Community Events
* Advertiser Information
* FCC & Public File Resources

## Technology

* Next.js
* TypeScript
* Tailwind CSS
* GitHub
* DigitalOcean App Platform

## Skills

Claude Code skills that guide work on the KAZM website. Each lives in `.claude/skills/`.

| Skill | Description |
| --- | --- |
| kazm-brand | Brand guardian. Station identity, visual direction, heritage, and radio-first principles. |
| radio-ux | Radio UX architect. Listener experience, information architecture, accessibility, and navigation. |
| programming | Program & music director. Format clock, dayparts, schedule, What's Playing, and the defined vs. situational radio model. |
| local-news | Local news editor. Community trust, public service, and useful regional journalism. |
| weather | Weather desk. Forecasts, conditions, alerts, heat safety, and Northern Arizona weather coverage. |
| sports | Sports editor. Regional and Arizona sports coverage, schedules, scores, and broadcasts. |
| community | Community editor. Local events, community information, and connection to Sedona and Northern Arizona. |
| public-service | Public safety & alerts desk. Emergency alerts, wildfire, road closures, advisories, and override authority. |
| advertising | Sponsorship & advertising director. Local advertiser relationships and listener-first sponsor content. |
| history | Station historian, archival preservation, community memory, and broadcast heritage. |

## Mission

To provide Northern Arizona listeners with trusted local information, community connection, and great music while preserving the heritage and local identity of KAZM Radio.
