# KAZM 106.5 FM & 780 AM

Mellow Mountain Radio

Serving Sedona, the Verde Valley, and Northern Arizona with Yacht Rock, local news, weather, sports, community information, and live radio programming.

## Features

* Live Audio Streaming
* Programming Schedule
* Local News
* Weather
* Sports Coverage
* Community Events
* Advertiser Information
* FCC & Public File Resources

## Technology

* Next.js
* TypeScript
* Tailwind CSS
* GitHub
* DigitalOcean App Platform

## Mission

To provide Northern Arizona listeners with trusted local information, community connection, and great music while preserving the heritage and local identity of KAZM Radio.
